# App Legal Pages

This public repository is prepared to host Legal, Privacy, and Support pages for apps.

## Apps

- RealQuest / 現実クエスト: public page scaffold is in preparation. Terms, Privacy Policy, and Support pages are not published yet.

## Source Layout

- `docs/`: future GitHub Pages source directory.
- `docs/realquest/`: planned RealQuest page area.
- `docs/realquest/index.md`: scaffold landing page for the future RealQuest page area.

## GitHub Pages Setup

GitHub Pages is intended to use branch `main` and folder `/docs`.

Codex does not change repository Pages settings. To publish the scaffold, a repository maintainer should:

1. Open `Settings` > `Pages` for this repository.
2. Set Source to `Deploy from a branch`.
3. Select branch `main` and folder `/docs`.
4. Save the setting and wait for deployment.
5. Confirm the scaffold pages load before adding store metadata URLs.

Expected scaffold URLs after Pages is enabled:

- `https://97jnjp-rgb.github.io/app-legal-pages/`
- `https://97jnjp-rgb.github.io/app-legal-pages/realquest/`

These scaffold URLs are not final Legal, Privacy, Support, or store metadata URLs.

## Publication Notes

- This repository does not contain the private RealQuest app source code.
- Placeholder-bearing draft policy files are not published here.
- Final Terms, Privacy Policy, and Support pages should be added only after final copy, contact, and effective date are approved.
- The current scaffold pages must not be used as final policy, privacy, support, or store metadata URLs.
- Do not add secrets, private contact credentials, signing files, tokens, analytics, forms, or tracking.
