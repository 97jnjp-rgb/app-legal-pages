# App Legal Pages

This public repository is prepared to host Legal, Privacy, and Support pages for apps.

## Apps

- RealQuest / 現実クエスト: public page scaffold is in preparation. Terms, Privacy Policy, and Support pages are not published yet.

## Source Layout

- `docs/`: future GitHub Pages source directory.
- `docs/realquest/`: planned RealQuest page area.

## Publication Notes

- This repository does not contain the private RealQuest app source code.
- Placeholder-bearing draft policy files are not published here.
- Final Terms, Privacy Policy, and Support pages should be added only after final copy, contact, and effective date are approved.
- Do not add secrets, private contact credentials, signing files, tokens, analytics, forms, or tracking.
